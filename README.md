A habit, task, journal & garden reward app.

## Getting Started

### Prerequisites
- Node.js 18+ (or 20+ recommended)
- npm, pnpm, or yarn

### Setup
```bash
npm install
```

### Run the dev server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see Bloom.

### Build for production
```bash
npm run build
npm run start
```

## MVP Notes
- Bloom stores tasks and garden flowers in `localStorage` for a single demo user.
- Use the **Reset today** button in the Tasks section to clear only today's data.
